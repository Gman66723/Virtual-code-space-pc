# Virtual-code-space-pc
Use GitHub code spaces to have a PC running ubuntu 22.04 latest

* # How to run
 * Fork
 * Create a codespace of this repo
 * Git clone this repo 
 * cd ./Virtual-code-space-pc
 * Run chmod +x ./start.sh
 * ./start.sh
 * Wait for vs code to tell you that theres a port running on XXXX
 * Then wait again for a seccond port notification but its on port 6080
 * Follow the link, it will look like an archive
 * select Novnc.html or lite.html
 * when it asks for passwd, the password is "password"
 * you can modify w sudo passwd
 * you should have a gui, and can open apps
